# Skill — Read project insumos

Goal: understand the full SIG-UTCUTS Chile project before coding.

Steps:
1. Read every `.md` file in `insumos/`.
2. Extract:
   - product objective;
   - modules;
   - entities;
   - API requirements;
   - UI requirements;
   - geospatial requirements;
   - prioritization rules;
   - MRV rules;
   - security rules;
   - acceptance criteria.
3. Create `docs/01_implementation_plan.md`.
4. The plan must include:
   - architecture;
   - folder structure;
   - MVP scope;
   - development sequence;
   - risks;
   - testing strategy.
5. Do not write application code until this artifact exists.
